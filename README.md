# TorCalculator

Локальный калькулятор для учёта перепродаж.

## Требования

- Python 3.10+
- Node.js 18+ (для сборки UI)
- pnpm или npm

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

## Режим разработки

```bash
python app.py --dev
```

## Сборка UI

```bash
cd ui
pnpm install
pnpm build
```

## Горячие клавиши

- `F5` или `Ctrl+R` - перезагрузка
- `F12` или `Ctrl+Shift+I` - devtools
